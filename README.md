识字打卡工具 - AI开发需求文档 V4
产品概述
一款帮助低龄儿童识字打卡的工具。家长管理字库，并录制答对/答错时的提示语音。孩子完成每日识字挑战，汉字读音由系统TTS自动朗读。支持家庭多设备随时使用，进度实时云端同步。单个孩子档案。

技术架构
前端（响应式网页）  →  云端服务器
      实时同步            │
                    数据库 + 音频文件存储
前端：Vue（响应式网页，适配PC和手机浏览器）
服务端：Node.js / Python 均可
数据库：MySQL 或 PostgreSQL
音频存储：服务端本地文件夹（如 /uploads/audio/）
汉字朗读：系统TTS（无需额外录音）
需要联网使用
数据结构
sql
word_library（字库）
id / character / pinyin / learn_date / add_time / is_active

checkin_records（打卡记录）
id / checkin_date / challenge_count / correct_count / wrong_count / total_rounds / duration

answer_details（答题明细）
id / record_id / character / is_correct / wrong_times / round_correct

prompt_audio（提示录音）
id / type / audio_url / create_time
-- type: correct（答对）/ wrong（答错）/ complete（完成）
后台管理功能
字库管理
批量输入汉字（空格分隔），自动补全拼音
手动设置学习日期
每条记录显示：汉字 / 拼音 / 学习日期
支持搜索、筛选
可编辑学习日期
删除为逻辑删除（is_active=0）
提示录音管理
家长录制三类提示语音
✅ 答对提示（例：哇好棒！答对了！）
❌ 答错提示（例：没关系，再想想！）
🎉 完成提示（例：今天表现真棒！）
每类可录制多条，答题时随机播放其中一条
按住录音 → 松开回放 → 确认保存或重录
支持重新录制覆盖

功能需求
首页
显示字库总数
挑战字数选择：10 / 15 / 20 / 自定义
自定义字数上限为字库总数
显示连续打卡天数和今日是否已打卡
开始挑战按钮
答题流程
从字库随机抽取指定数量的字
大字展示汉字
点击汉字 → 系统TTS朗读该字读音
两个按钮：✅ 认识 / ❌ 不认识
答对：撒花动画 + 播放家长录制的答对提示音
答错：轻柔动画 + 播放家长录制的答错提示音 + TTS再读一遍该字
顶部显示进度条和当前对错数
错题复习
一轮结束后展示所有错题
TTS逐个朗读错题
错题再答一轮
循环直到全部答对或用户选择跳过
打卡完成
显示本次成绩：挑战字数 / 一次通过数 / 练习后通过数 / 用时
播放家长录制的完成提示音
实时同步记录到云端
显示连续打卡天数
历史记录
日历视图显示打卡/未打卡
列表显示每次记录：日期 / 挑战字数 / 正确率
点击展开详情：一次通过的字 / 练习后通过的字
同步机制
每次进入自动拉取最新字库
答题进度本地暂存，答完后同步到云端
断网时可继续答题，恢复网络后自动上传记录
UI风格要求
整体风格：活泼卡通，色彩鲜艳
主色调：暖色系（橙/黄为主）
字体：大号圆体，清晰可爱
按钮：大圆角，有点击动画和音效
答对时：撒花/星星动画 + 欢呼音效
答错时：轻柔提示，不使用负面图标，保护孩子积极性
响应式设计：手机端优先，自适应PC大屏
后台管理页面：简洁清爽风格
